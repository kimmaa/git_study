git 처음 로그인 할때 
이름 설정
git config --global user.name "Your Name"
엔터
이메일등 연락처 설정
git config --global user.email "you@example.com"
엔터 

----
지금까지 커밋한 버전 보는 방법
git log 
엔터

---
git 수정된것 있는지 보는것: GUI의 변경내용 스테이징과 같은 것
git status

--
수정된 파일을 커밋 대기 상태에 넣는것 
git add 파일.확장자 
엔터

---
실제로 파일을 커밋하는 것
git commit -m "변경내용"
엔터

-----
commit 연습하기 좋은 날씨네
